## ABC Company Employee Dataset

## Overview
This repository contains a dataset of employees working in ABC Company. The dataset consists of employee details, including employee ID, name, position, age, salary, team, and other relevant information. The dataset provides insights into the workforce composition and distribution within the company.

## Dataset Details
- Number of Rows: 458
- Number of Columns: 9
### Columns
1. `employee_id`: Unique identifier for each employee.
2. `name`: Full name of the employee.
3. `position`: The position or job title of the employee.
4. `age`: Age of the employee.
5. `salary`: Salary of the employee.
6. `team`: The team to which the employee belongs.
7. `other_column1`: Brief description of other_column1.
8. `other_column2`: Brief description of other_column2.
9. `height`: Height of the employee (values have been corrected and standardized).

## Data Preprocessing
- Incorrect data in the `height` column has been fixed by replacing it with random numbers between 150 and 180.

## Analysis and Insights
The dataset has been analyzed to gain valuable insights into various aspects of the company's workforce. Some of the key findings include:

1. Distribution of Employees in Each Team and Position.
2. Segregation of Employees by Different Positions.
3. Age Group with the Highest Number of Employees.
4. Team and Position with the Highest Salary Spending.
5. Visualization of the Correlation between Age and Salary.
